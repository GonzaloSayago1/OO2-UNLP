# 📚 Orientación a Objetos 2 - UNLP

Repositorio dedicado a las resoluciones de las prácticas y ejercicios de la materia **Orientación a Objetos II** de la Facultad de Informatica, UNLP.

## 🛠️ Contenido del Repositorio
El repositorio está organizado por los grandes temas de la cursada:

* **Refactoring:** Aplicación de técnicas para mejorar la calidad del código sin alterar su comportamiento funcional.
* **Patrones de Diseño:** Implementación de soluciones estándar a problemas comunes (Strategy, State, Factory, Observer, etc.).
* **Frameworks:** Ejercicios utilizando herramientas vistas en la cátedra.

## 💻 Tecnologías Utilizadas
* **Lenguaje:** Java 17+
* **Gestor de Dependencias:** Maven
* **IDE:** Eclipse
* **Control de Versiones:** Git & GitHub
